Commands

print

Usage: print TEXT

Description: prints a text.

Example Script:

print Hello World!

waitforenter

Usage: waitforenter

Description: waits until the user presses Enter

msgbox

Usage: msgbox TEXT

Description: puts the text into an messagebox and shows it

Example Script:

msgbox Hello World!

errorbox

Usage: errorbox TEXT

Description: Similar to msgbox, it makes a messagebox window, but with an error sign.

Example Code:

errorbox An Error occured!

infobox

Usage: infobox TEXT

Description: Similar to msgbox and errorbox, it makes a messagebox window, but with an information sign.

download

Usage: download URL PATH

Description: Download a file.

Example Code:

download https://cfpweb.ddns.net/test.txt C:\Users\Public\Downloads\test.txt

presskeys

Usage: presskeys KEYS

Description: Presses Keys on the Keyboard

Example Code:

presskeys Hello World!

Bonus: You can also use special buttons: View the list

title

Usage: title TEXT

Description: Change the title

Example Code:

title Hello World!

setclip

Usage: setclip TEXT

Description: Change the clipboard content

Since: 1.7.0

Example Code:

setclip Hey (:! I replaced your clipboard!

print.

Usage: print.

Description: Print an empty line

Since: 1.7.0

waitforkey

Usage: waitforkey

Description: Similar to waitforenter, the programm waits for the user to press any key

Since: 1.7.0

Comments

'

The comment symbol of visual basic and maybe others.

//

The comment symbol of many languages

REM

The comment symbol of Batch

Variables

Variables work with ALL commands

Usage: %VARNAME%

Example Code:

print %hours%:%minutes%:%seconds% %year%/%month%/%day%
waitforend

hours

Will be replaced with the hours.

minutes

Will be replaced with the minutes.

seconds

Will be replaced with the seconds.

year

Will be replaced with the year.

month

Will be replaced with the month.

day

Will be replaced with the day.

cowversion

Will be replaced with the current version of cowscript.

pcname

Will be replaced with the name of the computer.
